# Screenshot Capture Guide for Chrome Web Store

**Purpose:** Instructions for capturing the 5 required screenshots for Chrome Web Store submission.

**Target Size:** 1280×800 px (primary) or 640×400 px (backup)  
**Format:** PNG, JPG, or GIF  
**Quality:** High resolution, clear UI elements, readable text

---

## Browser Setup for Screenshots

### Step 1: Prepare Your Screen

1. **Close distractions:**
   - Close other tabs in Chrome (keep only Clone2GHL open)
   - Close other applications
   - Disable Chrome extensions temporarily (notifications, etc.)

2. **Set browser zoom to 100%:**
   - Press Ctrl+0 (or Cmd+0 on Mac)
   - This ensures consistent scaling

3. **Set resolution/window size:**
   - Resize your Chrome window to approximately **1280×800 px**
   - Use DevTools to verify: F12 → Click device emulation if needed
   - Or use a tool like [Screenshot.guru](https://screenshot.guru/) for exact sizing

### Step 2: Test Extension

Make sure Clone2GHL is fully functional before capturing:
- Extension loads without errors
- Dashboard opens
- All tabs are accessible
- No console errors (F12 → Console)

## Screenshot 1: My Funnels Tab

**What to show:** Dashboard with cloned funnels grid

### Setup:
1. Open Clone2GHL Dashboard
2. Go to **My Funnels** tab
3. Ensure 3-4 sample funnels are visible in the grid
4. Show the URL input field at the top
5. Show the plan badge (FREE/PRO/AGENCY) in left sidebar

### Composition:
- Left sidebar (30% of screen): Shows navigation tabs, plan badge
- Main content (70%): Shows "My Funnels" grid with 3-4 Sample funnel cards
- Each card shows: thumbnail image, funnel name, clone date
- Top of page shows: URL input field, "Clone Page" button

### How to capture:

**Option A: Use Chrome DevTools (Accurate)**
1. Open Clone2GHL Dashboard
2. Press **F12** to open DevTools
3. Press **Ctrl+Shift+P** → type "Screenshot"
4. Select "Capture full page screenshot"
5. DevTools saves the screenshot automatically
6. Crop to 1280×800 px if needed

**Option B: Use Windows Snipping Tool**
1. Press **Shift+Ctrl+S** (or Win+Shift+S)
2. Drag to select the area (≈1280×800)
3. Save as PNG to `/extension/assets/screenshots/screenshot_1_my_funnels.png`

**Option C: Use Online Tool**
1. Go to [Screenshot.guru](https://screenshot.guru/)
2. Paste dashboard URL (if locally hosted, won't work)
3. Download screenshot
4. Crop to 1280×800 px

### File name:
`screenshot_1_my_funnels.png`

### Checklist:
- [ ] Size is 1280×800 px
- [ ] UI elements are sharp and readable
- [ ] My Funnels tab is active/highlighted
- [ ] At least 3-4 funnel cards visible
- [ ] Plan badge visible in sidebar
- [ ] No personal/sensitive data visible
- [ ] No other browser tabs visible

## Batch Screenshot Capture Script (Advanced)

If you want to automate screenshots, you can use this browser automation:

### Using Puppeteer (Node.js)

```javascript
const puppeteer = require('puppeteer');

(async () => {
  const browser = await puppeteer.launch({
    headless: false,
    args: ['--start-maximized']
  });
  
  const page = await browser.newPage();
  
  // Set viewport to 1280x800
  await page.setViewport({width: 1280, height: 800});
  
  // Screenshot 1: My Funnels
  // (Navigate to dashboard & My Funnels tab)
  await page.screenshot({
    path: './screenshots/screenshot_1_my_funnels.png'
  });
  
  // Screenshot 2: AI Tools
  // (Navigate to AI Tools tab)
  await page.screenshot({
    path: './screenshots/screenshot_2_ai_tools.png'
  });
  
  // ... repeat for other screenshots
  
  await browser.close();
})();
```

## Converting Browser Screenshot to 1280×800 px

If your screenshot is a different size, resize it:

### Using Windows

**Option A: Use Paint**
1. Open Paint (Ctrl+1 or search "Paint")
2. Open your screenshot
3. Select Image → Resize
4. Set to 1280 width × 800 height
5. Save as PNG

**Option B: Use Image Editor (Recommended)**
- Download ImageMagick or use online tools like [Pixlr.com](https://pixlr.com/)
- Resize image to 1280×800 px
- Export as PNG

**Option C: Command Line (ImageMagick)**

```bash
# Install ImageMagick if needed
# Then run:
magick convert screenshot_1.png -resize 1280x800 screenshot_1_resized.png
```

## Common Issues & Fixes

**Issue:** Screenshot is blurry or pixelated
- **Solution:** Ensure browser zoom is 100% (Ctrl+0), increase resolution

**Issue:** Text is too small to read
- **Solution:** Screenshot might be the wrong size, or text is genuinely small; consider zooming or resizing

**Issue:** Chrome DevTools/sidebar visible in screenshot
- **Solution:** Press F12 to hide DevTools before capturing

**Issue:** Extension not visible or working during screenshot
- **Solution:** Reload extension (chrome://extensions), restart browser, check for errors
